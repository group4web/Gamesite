Game Description
Breakout The object of this game is to break all the bricks containing you. You will have 3 lives to try and breakout. Breakout is only available on laptops, and PC’s due to the fact that it requires a keyboard. However, it will be coming soon to more platforms with new levels, and more speed!
Jack-Dash This game is a side-scroller where the player controls a pink block, attempting to fly between pillars of doom without hitting them. You control the block by clicking or holding the UP button. This game is playable on mobile phones, laptops, and PC’s.
Fill'em Up The object of this game is to try and match both shapes to be the same size! Sadly, Fill’em Up is only available on laptops, and PC’s due to the fact that it requires a keyboard. However, it will be coming soon to more platforms with new, more complicated shapes!
